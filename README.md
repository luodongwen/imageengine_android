# 图像处理/人像美化引擎Android版

#### 介绍
zphotoengine升级版，包含照片美化和人脸美化两部分，使用非常简单，直接aar包调用即可

#### 使用说明
1. 商用级别，速度快，效果好
2. imageengine_aar文件夹下面为图像处理/人脸美化aar包，里面包含了PS处理/滤镜/美颜等相关的api调用
3. imageengine_sample为可用的android调用demo
